# Netflix Data Analysis

## Problem Statement
The objective of this project is to analyze Netflix’s movies and TV shows dataset to identify trends in content type, release patterns, country-wise distribution, ratings, and duration using data cleaning and exploratory data analysis techniques.

## Dataset
Source: Kaggle – Netflix Movies and TV Shows  
Records: 8,800+ titles

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Netflix has more Movies than TV Shows.
- Content additions increased significantly after 2015.
- The United States, India, and the United Kingdom dominate content production.
- Most content is rated TV-MA.
- Most movies are between 80–120 minutes long.

## Conclusion
This analysis demonstrates how data cleaning and exploratory data analysis can be used to derive actionable insights from real-world entertainment data.
